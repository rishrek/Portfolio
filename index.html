PROJECT // 02
🎹

PCB Electronic Piano

PCB Electronics Design

An engineered electronic piano built entirely on a custom-routed PCB, utilizing NE555 timer ICs in astable configuration to generate precise, mathematically-derived audio frequencies for each key. The design incorporates multi-stage RC networks for accurate tone calibration, manual layout routing for signal integrity, and a clean schematic hierarchy — demonstrating mastery of analog circuit theory translated into a physical, playable instrument.
